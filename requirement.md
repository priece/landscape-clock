# 横屏模拟时钟
- 横屏模拟时钟是一个基于kotlin的应用程序，它可以在横屏模式下显示模拟时钟，运行在android设备上。
- 该应用程序的主要功能是显示当前时间，并且仅在横屏模式下进行查看。
- UI效果
  - 左侧一个模拟时钟，有表盘，时分秒指针。
  - 右侧一个月历，显示当前月份的日期。
- UI参考图：./resource/clock-ui.png
- 技术实现
  - 该应用程序使用kotlin语言编写，基于android studio开发环境。
  - 该应用程序的主要代码文件包括：
    - MainActivity.kt：主活动类，负责初始化UI和处理用户交互。
    - ClockView.kt：模拟时钟视图类，负责绘制时钟表盘和指针。
    - CalendarView.kt：月历视图类，负责绘制月历和日期。
    - 最终打包成apk文件，安装在android设备上运行。
